Spring Boot Consul Registry Server
====================================

### Developement

* docker-compose up -d 
* visit http://localhost:8500/ui/


### Consul Configuration

Consul Key/Value支持的configuration目录结构如下：

    config/testApp,dev/
    config/testApp/
    config/application,dev/
    config/application/
